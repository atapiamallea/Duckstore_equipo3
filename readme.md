:Duckstore_equipo3
==========================


## Features

- Landing Page (Home)
- Página de catálogo 
- Página de detalle
- Página de contacto

##Sección de Formulario
Se ha integrado un formulario de contacto profesional que incluye:
- **Campos:** Nombre, Email, Asunto y Mensaje.
- **Validación:** Uso de atributos `required` de HTML5.
- **Interactividad:** JavaScript captura el evento *submit*, previene la recarga y muestra los datos mediante `console.table()` para una mejor visualización técnica.


## Información General
Se desarrollará una página web estática para el e-commerce DuckStore
La interfaz, gracias a sus colores vívidos y gráficas divertidas, invita el usuario a descubrir las diferentes categorías de patos de goma
Las categorías, de momento, son “Animales” y “Súperheroes”
En la sección “Adopta un patito”, se enseñará el último producto lanzado
El usuario podrá contactar directamente con la empresa pulsando el botón “Contactanos”
